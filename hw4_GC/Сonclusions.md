Сравнение работы G1 и Serial GCs
-------------------------------
Графики памяти:

![G1 GC](Сonclusions/G1GC.jpg)

![Serial GC](Сonclusions/SerialGC.jpg)

Сравнение операций:

|               | G1 GC        | SerialGC            |
| :-----------: | :----------: | :-----------------: |
| Общее время   | 155.3 ms     | 4 min |
| Среднее время | 1.233 ms     | 144.6 ms              |
| Мин. время    | 0.767 ms     | 0.062 ms              |
| Макс. время   | 2.396 ms     | 378.765 ms              |


Вывод:
> Из сравнения операций можно сделать вывод, 
> Время затраченное G1 GC на выполнение очистки, сильно отличается, в меньшуюю сторону, от SerialGC.
> Можно сделать вывод, что при использовании многоядерных и много процессорных машин. 
> Использование G1GC как сборщик мусора, будет лучшим выбором для приложений чувствительных к паузам.
