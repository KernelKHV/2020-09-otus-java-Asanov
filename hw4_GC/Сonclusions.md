Сравнение операций:

|                 | G1 GC        | SerialGC            |
| :-------------: | :----------: | :-----------------: |
| Кол-во операций | 12           | 9				   |
| Общее время     | 4255 ms      | 5539 ms 		       |
| Среднее время   | 384.85 ms    | 615.4 ms            |
| Мин. время      | 1.693 ms     | 59.913 ms           |
| Макс. время     | 619.937 ms   | 673.721 ms          |

 


Вывод:
> Из сравнения операций можно сделать вывод, 
> Время затраченное G1GC на выполнение очистки, сильно отличается, в меньшуюю сторону, от SerialGC. 
> Приложении c ним работало дольше, а сам GC работал меньше и запускался чаще чем, SerialGC.
> Можно сделать вывод, что использование G1GC будет лучшим выборов для приложения чувствительного к паузам в работе, 
> а также если существует проблема падения приложения по OOM. Вероятно это связано с тем, что часть работы G1GC 
> выполняет паралельно общему потоку, задействуя при этом больше ресурсов процессора(видно на графике).
